What you actually need.

Nexus runs entirely on local AI with Ollama. Most keys below are completely optional — notifications and cloud fallback only.

Required
Ollama (local runtime)
The local AI engine that runs all models on your computer. No API key — just an app install. Runs Qwen, DeepSeek, and Llama locally for free.
ollama.com — free download, no account needed
100% Free
No OpenAI / Anthropic key
Nexus uses open-weight models only: Qwen 2.5, DeepSeek R1, Llama 3.2, all running locally via Ollama. You never pay per token or per query.
Zero API cost — all inference runs on your machine
Optional
Telegram Bot Token
Enables approval notifications when Nexus drafts an email or books a meeting. You approve, edit, or reject directly from Telegram. Get one free from @BotFather in 2 minutes.
Telegram → search @BotFather → /newbot → copy token
Optional
Groq API Key
Cloud fallback only. If your computer is slow or Ollama is busy, Nexus can fall back to Groq's free tier (Llama 3.3 70B). Groq's free tier has generous limits. Not needed for normal use.
console.groq.com → API Keys → Create key (free, no card)
Optional
Gmail OAuth Credentials
Required only if you want Nexus to draft and send emails. It reads your past emails to match your writing tone, then drafts messages for your approval before sending anything.
console.cloud.google.com → Gmail API → OAuth credentials
Not needed
OpenAI / Claude / Gemini
Nexus uses open-weight models running locally. You do not need OpenAI, Anthropic, or Google AI keys. The system is deliberately designed to work without any of these.
No subscription, no billing, no per-token cost
$0
Summary: start with zero keys
Install Ollama, run the setup script, and Nexus works immediately. Add Telegram later for approval notifications. Everything else is optional.

architecture

Seven stages. One answer.

Every query flows through the same pipeline — no shortcuts, no hallucinations, no unjustified confidence.

01
Input Guard & Classification
Every query is scanned for prompt injection before any agent sees it. The classifier routes it to the right pipeline in under 1 second using a local Llama model.
injection-scanllama3.2:3blive_data | action | knowledge
02
Live Data — 6 Browser Agents in Parallel
For price, flight, weather and market queries: 6 Playwright browser agents scrape independent sources simultaneously. One CAPTCHA-blocked? A fallback source takes its place.
playwrightstealth-mode6-parallelauto-retry
03
5-Layer Validation & Cross-Verification
Each scraped result passes freshness, format, outlier, DOM integrity, and trust-rank checks. A weighted consensus is computed. The LLM only ever receives this verified JSON — never raw web data.
grounding-gateweighted-consensusoutlier-detection
04
Agent Meeting Room
Researcher, Reasoner, Critic, and Fact-checker enter a shared debate. The Critic specifically challenges unsupported claims. Rounds continue until agents converge or the maximum is reached.
qwen2.5:72bdeepseek-r1:32badaptive-roundsconvergence-check
05
Decision Agent
Reads the full transcript of every debate round. Scores each agent on evidence quality and claim stability. Cross-checks every number against the original scraped data. Renders the final verdict.
full-transcriptagent-scoringsource-crosscheck
06
HITL Gate for Actions
For irreversible actions (send email, book flight), the system stops and sends a draft to your Telegram for approval. Nothing executes until you say yes. Auto-cancelled after 24 hours of inaction.
telegram-approval24h-expirytone-matchingmulti-account
07
PII Masking & Output
The final answer passes through Presidio PII masking (removes names, phone numbers, Aadhaar, PAN) and an output sanitizer. Then it's delivered with source citations and confidence score.
presidiopii-maskedcited-sourcesconfidence-score
installation

Running in minutes.

Works on Windows, Mac, and Linux. Windows users get a one-click installer.

Step 1 — Download and run the installer
# 1. Install Python 3.11+ from https://python.org # IMPORTANT: Check "Add Python to PATH" during install # 2. Download Nexus and run the Windows installer git clone https://github.com/your-repo/nexus-ai cd nexus-ai install-windows.bat
Step 2 — Start Nexus any time
# Double-click this file to start Nexus start-windows.bat # Or from PowerShell .\start-windows.ps1
macOS setup
# Install Ollama brew install ollama # Pull required models ollama pull llama3.2:3b ollama pull nomic-embed-text ollama pull qwen2.5:7b # Install and start Nexus git clone https://github.com/your-repo/nexus-ai cd nexus-ai pip install -e ".[dev]" python scripts/setup.py ollama serve & uvicorn src.api.main:app --host 127.0.0.1 --port 8000
Linux setup
# Install Ollama curl -fsSL https://ollama.com/install.sh | sh # Pull required models ollama pull llama3.2:3b ollama pull nomic-embed-text ollama pull qwen2.5:7b # Install and start Nexus git clone https://github.com/your-repo/nexus-ai cd nexus-ai pip install -e ".[dev]" python scripts/setup.py ollama serve & uvicorn src.api.main:app --host 127.0.0.1 --port 8000
